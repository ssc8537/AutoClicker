## 阶段 2 测试：脚本管理系统

### 测试目标
用户可以通过 JSON 配置文件创建、编辑和管理按键脚本。每个脚本可独立设置热键、触发模式、执行次数、速度倍率。

### 脚本 JSON 格式示例

  {
    "name": "Hello World",
    "key1": 120,
    "key2": 0,
    "mode": 0,
    "count": 1,
    "speed": 1.0,
    "state": true,
    "script": "h 100 e 50 l 50 l 50 o 200 space 100 w 100 o 50 r 50 l 50 d 0"
  }

### 参数说明
| 字段 | 说明 | 可选值 |
|------|------|--------|
| name | 脚本名称 | 任意文本 |
| key1 | 主热键（虚拟键码） | 120=F9, 121=F10... |
| key2 | 辅助热键（0=无） | 162=LCTRL, 160=LSHIFT... |
| mode | 触发模式 | 0=切换, 1=按住, 2=松开 |
| count | 执行次数 | 0=无限, 1~9999 |
| speed | 速度倍率 | 0.1~10.0 |
| state | 启用/禁用 | true/false |

### 测试清单

| # | 操作 | 预期结果 | 打勾 |
|---|------|---------|:----:|
| 1 | 在 macro/ 目录创建 Hello World JSON 脚本 | 文件能被程序加载 | [ ] |
| 2 | 设置 key1=120(F9), mode=0(切换), count=1 | 按一次 F9 打一轮 Hello World | [ ] |
| 3 | 设置 mode=1(按住), count=0(无限) | 按住 F9 时持续循环，松开停止 | [ ] |
| 4 | 设置 mode=2(松开), count=1 | 按下 F9 不触发，松开时打一轮 | [ ] |
| 5 | 设置 count=3 | 按一次 F9，Hello World 重复 3 次 | [ ] |
| 6 | 设置 speed=2.0 | 打字速度翻倍（延迟减半） | [ ] |
| 7 | 设置 speed=0.5 | 打字速度减半（延迟加倍） | [ ] |
| 8 | 设置 state=false | 该脚本不触发（其他脚本正常） | [ ] |
| 9 | 设置 key1=121(F10) + key2=162(LCTRL) | 按 Ctrl+F10 触发 | [ ] |
| 10 | 两个脚本绑定同一热键 | 程序提示冲突 | [ ] |
