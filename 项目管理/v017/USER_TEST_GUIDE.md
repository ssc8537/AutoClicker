## 新手上路：第一次怎么用这个程序

> 如果你是第一次接触这个项目，请按以下顺序操作。

### 第一步：找到程序

程序在电脑的这个位置：
```
桌面 > 1-test > 1-自动连点器 > my-automation-tool > main.py
```
完整路径：`C:\Users\s\Desktop\1-test\1-自动连点器\my-automation-tool\main.py`

### 第二步：启动程序

**方法一（最简单）**：双击 `main.py` 文件
**方法二**：在终端（PowerShell）中运行：
```
cd C:\Users\s\Desktop\1-test\1-自动连点器\my-automation-tool
python main.py
```

启动后你会看到一个窗口，标题是 "MyAutoPlayer - 键盘自动化序列播放器"。
窗口顶部有一行大字，默认显示红色文字："[红] 热键已禁用"

### 第三步：窗口长什么样

```
+-------------------------------------------------------+
| MyAutoPlayer - 键盘自动化序列播放器                      |
+-------------------------------------------------------+
|                                                       |
|         [红] 热键已禁用  或  [绿] 热键已启用            |
|                                                       |
|          MyAutoPlayer 已启动                           |
|                                                       |
|          按 F12 启用热键                               |
|          按住 F9 在记事本中打出 hello world             |
|          （鼠标不要在程序窗口内）                        |
|                                                       |
+-------------------------------------------------------+
```

### 第四步：按键说明

| 按键 | 功能 | 备注 |
|------|------|------|
| F12 | 切换热键启用/禁用 | 按一次启用，再按一次禁用。任何时候都能按 |
| F9 | 按住循环执行小写 hello world；松开停止 | 只有启用状态下、鼠标不在窗口内时才能触发 |
| Alt+F4 | 关闭窗口 | 关闭后自动清理钩子，不会有残留 |

### 第五步：测试前准备

1. 打开一个**记事本**（按 Win 键，输入"记事本"）
2. 把记事本窗口和程序窗口**并排摆放**，两个窗口都能看到
3. 确认程序窗口顶部显示**红色**的"热键已禁用"

### 测试失败了怎么办？

| 现象 | 可能原因 | 解决办法 |
|------|---------|---------|
| 双击 main.py 没反应 | Python 没装好，或没关联 .py 文件 | 用方法二：终端运行 python main.py |
| 窗口一闪就没了 | 代码有报错 | 告诉 AI "窗口一闪就没了"，AI 会查日志 |
| 按 F12 没反应 | 被其他软件占用了 F12 | 关掉录屏软件、浏览器等再试 |
| 按 F9 没出 hello world | 热键还在禁用状态，或鼠标在窗口内 | 确认顶部文字是绿色，且鼠标不在窗口中 |
| 关闭窗口后任务管理器还有 python.exe | 钩子没清理干净 | 终端运行：Get-Process -Name python \| Stop-Process -Force；告诉 AI 这个情况 |

## 已知注意事项
1. 管理员权限：部分游戏需要以管理员身份运行程序才能用 SendInput 发送按键
2. 被其他程序占用热键：如果 F9/F12 没反应，可能被其他软件（如录屏软件）占用了
3. 程序没反应时：打开任务管理器，杀掉所有 python.exe 进程，重启程序
4. 残留钩子紧急清理：在终端运行 Get-Process -Name python | Stop-Process -Force
5. 查看运行日志：日志文件在 my-automation-tool/logs/app.log，用记事本打开
