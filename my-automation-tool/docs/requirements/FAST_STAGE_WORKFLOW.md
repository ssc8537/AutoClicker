# 快速阶段模板

每一个新能力只建立这一页规格，不再复制多个编号计划或员工报告。新接手默认只读根 `AGENTS.md` 与 `CURRENT_HANDOVER.md`；只有实际开始该能力才打开本页。

```text
阶段：
目标（用户能看到什么）：
范围外：
已锁定行为与安全/回退：
自动检查：
Windows 验收步骤与通过回复：
READY 依据：
```

负责人将上述内容填入当前阶段规格后，直接实现、自动验证、给出中文教程并等待用户验收。用户通过后，`CURRENT_HANDOVER.md` 只保留当前状态、测试证据、未覆盖风险和唯一下一步；历史编号交接仅作档案，不再默认阅读。
