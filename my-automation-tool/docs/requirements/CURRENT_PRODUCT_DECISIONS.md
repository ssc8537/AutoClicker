# 当前产品决定

触发页是实时设置页，不要求用户编辑宏代码：热键、模式（切换/按下）、次数、速度、启用状态均直接修改并保存。热键可为单个键盘键或鼠标侧键1/2；F12 永远保留为全局启停。切换为按一次启动、再按一次停止；按下为按下启动、松开立即停止。相同按键的所有启用宏并发运行，每个宏独立停止；状态列可逐项点击启用/禁用。

不复制优秀案例源码、资源或宏格式；本项目仍只运行可信本地 Python 宏。Stage 6B 等待 Windows 验收，默认不提交、不推送。

触发表左侧名称列优先占宽；状态列固定窄宽，且只显示“启用”“禁用”或“错误”。右侧标题为“触发详情（自动保存）”。

状态列可直接点击任意一行，不需要先选中；每行立即保存自己的启用状态。触发录入支持案例命名的键盘键，以及左键、右键、中键、侧键1、侧键2；F12 仍保留为全局启停。次数和速度仍使用本项目既有安全范围，但其上下按钮必须可点击。滚轮自动化、窗口绑定和识别不在范围内。
