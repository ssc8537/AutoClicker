# 当前产品决定

## 已验收能力（2026-07-18）

- 可信本地 Python 是唯一宏格式：元数据加 `run(player)`；仅一个活动宏，F9 执行、F12 全局启停、F2 仅 UI 占位。
- 宏可创建、编辑、保存、改名、重新加载和回收站删除；静态 AST 校验与原子保存/回退保持有效。
- 默认键位 `1/2/3/E/Q/R/Space/F` 可在设置页保存；中文语义 API 只发送这些物理键，不识别游戏状态。
- 鼠标只支持当前位置左右键按下/松开/单击/有限重复；重复为 1–100 次、真实间隔不少于 10ms，宏结束、异常、F12 和退出都会释放。
- Stage 4T 已 Windows 验收：粉色标题栏的 `×` 安全退出、`—` 任务栏最小化、`藏` 隐藏到托盘；托盘菜单显示/隐藏/退出，单击恢复。程序/托盘共用自制粉色 SVG 图标，待一次 Windows 图标目测。

## 永久边界

不实现 JSON/QIM、录制、坐标/滚轮/图像或游戏识别、多宏并行、新热键、开机自启或 AI 提示词更新，除非用户另开一个明确小阶段。优秀案例只读，禁止复制源码、资源、图标或品牌。默认不提交、不推送；用户宏文件不得覆盖、执行或提交。
