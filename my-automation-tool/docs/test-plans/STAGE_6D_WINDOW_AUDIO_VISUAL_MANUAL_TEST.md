# Stage 6D · 窗口、箭头、预览与提示音验收

> **安全准备：不打开游戏。** 先完全退出旧版“自动连招”，再启动下面的独立验收版。

启动文件：`dist-preview\MyAutoPlayer-Stage6D\MyAutoPlayer-Stage6D.exe`

| 步骤 | 你要做什么 | 正确结果 |
|---:|---|---|
| 1 | 打开记事本并让它挡住桌面，然后双击验收版。 | “自动连招”立即出现在记事本前面。随后点击记事本，记事本仍可正常到前面；程序不是永久置顶。 |
| 2 | 打开“触发”页，点选一个有效宏。看“次数”和“速度”的右边。 | 每个数字框右侧都有清晰的**酒红上三角、下三角**；上箭头增加、下箭头减少。测试后调回原值。 |
| 3 | 打开“宏库”，把鼠标移到任意宏名称行上，再点击该行。 | 悬停和选中都是很浅的天空海洋蓝，宏名称仍是深色、容易看清；没有深蓝底或白字看不清的问题。 |
| 4 | 打开“设置”，勾选“启用提示音（默认关闭）”。按 F12 启用、再按 F12 禁用；可再启动一个安全宏并停止。 | 勾选前没有声音；勾选后 F12 与宏开始/停止各有短促提示音。声音播放不影响 F12、宏停止或退出。 |

## 通过标准

四步都符合“正确结果”，即为通过。

## 小提示

- 提示音设置会保存；不喜欢声音时，取消勾选即可。
- 若旧程序仍在运行，先在任务栏或托盘退出它，再启动验收版，避免两个程序同时监听热键。
