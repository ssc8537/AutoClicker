# V023 四页视觉对齐：Windows 人工验收

> 前置：自动回归 23/23、`compileall` 与 `git diff --check` 已通过；本清单只验证真实 Windows 外观与既有安全回归。

## 准备

1. 将 Windows 显示缩放设为 100%，关闭本程序的旧实例。
2. 在 `my-automation-tool` 目录运行 `python main.py`；启动后保持 F12 默认禁用。

## 四页外观与窗口

1. 确认窗口宽度不可横向改变，可纵向调整，且不能缩到低于 510 高度。
2. 依次打开“宏库、触发、功能、设置”：顶部顺序不变，Candy 粉红色一致，无重叠、裁切或滚动条异常。
3. 宏库与触发页：确认左侧表格可用、右侧窄栏对齐；所有新建、编辑、保存、刷新、导入、导出、删除和保存触发设置均灰置不可用。
4. 功能页：确认说明文字清晰，鼠标/F2 相关控件仍灰置；F2 只显示占位，不可操作。
5. 设置页：确认左侧说明、右侧运行状态和只读设置清晰；状态文字、F12、OSD 与主题说明没有裁切。

## 既有安全回归（记事本）

1. 打开记事本并把焦点放在编辑区；按 F12，确认状态/OSD 从禁用变为启用。
2. 按既有宏方式触发 F9，确认现有 Python 宏行为正常；再按 F12 停止。
3. 按 F2，确认没有任何输入或热键行为。
4. 关闭 MyAutoPlayer 后，再按 F9/F12，确认程序不再响应热键。

## 反馈格式

- 通过：`V023 Windows 外观与 F9/F12/F2 安全回归通过`。
- 不通过：注明页面、控件、窗口高度和可见现象；不要自行修改代码。
