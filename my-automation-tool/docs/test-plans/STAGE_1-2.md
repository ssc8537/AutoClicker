## 核心思想

用"正确打出 Hello World"来测试每个阶段的功能是否实现。
每个阶段测试通过 = 软件该阶段功能正确。

## 阶段 1 测试：Hello World + 安全机制（已完成）

### 测试目标
程序启动后默认为禁用状态，按 F12 启用后，按 F9 在记事本中打出 Hello World。

### 测试清单

| # | 操作 | 预期结果 | 打勾 |
|---|------|---------|:----:|
| 1 | 启动程序 | 窗口顶部显示红色"热键已禁用" | [x] |
| 2 | 光标放记事本，直接按 F9 | 不打字（禁用状态有效） | [x] |
| 3 | 按 F12 启用 | 顶部变绿色"热键已启用" | [x] |
| 4 | 光标放记事本，按住再松开 F9 | 按住时由一个受控循环输出；松开后立即停止且不再继续输出 | [x] |
| 5 | 鼠标放在程序窗口内，按 F9 | 不触发（焦点检测有效） | [x] |
| 6 | 按 F12 禁用，按 F9 | 不再打印（禁用恢复） | [x] |
| 7 | 关闭程序，检查任务管理器无 python 残留 | 清理干净 | [x] |
