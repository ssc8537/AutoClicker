# Stage 5P：Windows 便携版验收教程

## 准备

1. 关闭所有旧的“我的自动播放器”或 Python 程序窗口。
2. 打开 `C:\Users\s\Desktop\1-test\1-自动连点器\my-automation-tool\dist\MyAutoPlayer`。
3. 不打开游戏；本教程只验证桌面程序和记事本。

## 操作与预期

1. 在资源管理器查看 `MyAutoPlayer.exe`。预期：文件图标是粉色爱心勾选，不是 Python 图标。
2. 双击 `MyAutoPlayer.exe`。预期：没有黑色命令行窗口；程序左上名称为“自动连招”，任务栏和托盘也是同一枚粉色图标。若看到黑色窗口，说明启动的是 `python main.py` 或旧 EXE，必须关闭后重新双击本目录 EXE。
3. 点击 `—`、`藏`、托盘图标和 `×`。预期：最小化、隐藏/恢复和真正退出仍与 Stage 4T 一致。
4. 重新启动 EXE，在“宏库”新建一个测试宏或编辑副本；在“设置”保存一个你容易识别的键位。退出 EXE 后再次启动。预期：宏文件与键位修改仍在 EXE 同级 `macros`、`config` 目录中，不会丢失。
5. 在记事本中分别启用 `Hello World`、`goodbye` 或 `新宏6` 中任一宏，按 F9 启动后再停止或等待自然结束。预期：绿色 OSD 显示“该宏名 宏运行中”，红色 OSD 必须显示同一个“该宏名 宏已停止”，绝不固定显示 Hello World；F12 启停、F2 无实际动作，不留下卡住的按键或鼠标。

## 通过回复

`Stage 5P 便携版、自动连招名称、无黑窗、粉色图标、宏/键位持久化、OSD 宏名和 F9/F12/F2 通过`
