# Stage 4T 系统托盘与窗口按钮：已验收，粉色图标复查

## 准备

1. 不打开游戏；关闭旧的 MyAutoPlayer 窗口。
2. 在 PowerShell 执行：

```powershell
cd C:\Users\s\Desktop\1-test\1-自动连点器\my-automation-tool
python main.py
```

3. 确认左上名称是“我的自动播放器”，窗口、任务栏和 Windows 通知区域均显示同一枚粉色爱心勾选图标；右侧仍有三个粉红按钮：`藏`（隐藏）、`—`（最小化）、`×`（退出）。若图标被折叠，请点击任务栏右侧 `^` 查看。若仍显示旧 Python 图标，请先确认旧进程已退出后重启一次；Windows 可能短暂缓存图标。

## 操作与预期

1. 按 F12 使状态变为绿色“热键已启用”。点击 `—`。预期：桌面上不再显示窗口，但任务栏仍保留程序按钮；从任务栏恢复后，绿色状态不变。
2. 点击 `藏`。预期：桌面和任务栏都不再显示程序窗口，但托盘图标仍在；单击托盘图标，窗口恢复到前台，F12 状态不变。
3. 右键托盘图标。预期：菜单只有“显示主窗口”“隐藏主窗口”“退出程序”。依次验证“隐藏主窗口”和“显示主窗口”能隐藏、恢复窗口，且不会改变 F12 状态。
4. 把鼠标放在窗口底边的细条上，光标应变为上下缩放；上下拖动。预期：窗口只能改变高度，宽度保持不变。
5. 再次按 F12 启用后，点击 `×` 或托盘菜单“退出程序”。预期：程序直接退出，窗口和托盘图标消失；不会仅隐藏到托盘。重新启动后，F2 仍无实际动作，F9/F12 仍按既有规则工作。

## 通过标准

用户已确认原五步均符合预期；本次只需确认中文名称以及窗口/任务栏/托盘的粉色图标一致。通过后回复：`中文名称和粉色窗口/托盘图标通过`。

## 未覆盖风险

本教程不测试游戏输入、鼠标坐标/滚轮、开机自启、通知、窗口位置持久化、录制、图像/游戏识别、多宏并行或新热键。若 Windows 系统托盘本身不可用，隐藏按钮会禁用，程序仍可正常退出和最小化。
