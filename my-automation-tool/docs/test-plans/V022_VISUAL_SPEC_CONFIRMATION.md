# V022 视觉规格确认清单（用户）

> 这不是运行测试。V022 没有修改程序、热键或输入行为，因此无需启动宏，也不要重新测试 F9、F12 或 F2。

## 你需要查看的材料

1. `../requirements/V022_VISUAL_ALIGNMENT_SPEC.md`。
2. 项目副本 `../reference/quickinput-ui/01-macro-library.png` 至 `04-settings.png`。
3. 当前 v021.1 程序界面（仅作为现状参考）；粉红主题以规格为准，不以图片中的旧版浅蓝替换。

## 请逐项确认

- [ ] 保留 Candy 粉红色、微软雅黑和四个顶部标签。
- [ ] 宏库采用“主表格 + 窄右侧操作栏”的层级；操作仍全部灰置。
- [ ] 触发采用“左侧概览 + 右侧只读详情”的层级；固定 F9 不显示为可编辑。
- [ ] 功能页保持稀疏留白；鼠标、窗口和时间能力仍清楚标为“后续阶段”。
- [ ] 设置页用“左说明 + 右固定值/禁用项”；F2 显示未注册，F12/OSD 不变成鼠标开关。
- [ ] 窗口继续固定宽度、最小高度 510，只能上下缩放；不采用半高裁切。
- [ ] 若你希望“更漂亮”的具体边距、标题栏、按钮或字体粗细与本规格不同，请逐项写出；否则回复“V022 视觉规格确认通过”。

## 确认后的流程

你的确认只授权 RequirementCertifier 评估未来 UI 实现是否 READY。真正修改 UI 后，项目会提供新的离屏自动测试、Windows 视觉对照和记事本 F9/F12/F2 安全回归教程。
