# Stage 2C-S：三角色切人规则 Windows 验收教程

> 本轮只验收 AI 提示词和作者说明的文字规则，不运行真实游戏输入，不测试鼠标、键位配置、触发页或系统托盘。

## 准备

1. 在 `my-automation-tool` 文件夹打开 PowerShell。
2. 运行：`python main.py`
3. 准备记事本，用于粘贴并阅读提示词。

## 步骤与预期

| 步骤 | 操作 | 预期结果 |
|---|---|---|
| 1 | 点击宏页右栏“AI 提示词”。 | 开头有 1号角色名称、2号角色名称、3号角色名称和三角色连招需求填写区。 |
| 2 | 阅读切人规则与示例。 | 同时列出六个方向；普通切人前为 60ms，任意回切为 1100ms；示例仍使用数字 1、2、3。 |
| 3 | 点击“复制提示词”，粘贴到记事本。 | 文本要求 AI 写角色编号注释，但不允许 `player.角色切换()` 等未发布接口。 |
| 4 | 关闭窗口并按原有方法检查 F12 启停和 F9。 | 提示词不会执行宏；既有热键和宏状态不变。 |

## 通过与失败反馈

通过时回复：`Stage 2C-S 三角色切人规则和 F9/F12 回归通过`。

失败时回复：`Stage 2C-S 第 X 步失败；缺失的方向/等待/角色名称：…；F12/F9 是否异常：是/否`，方便时附截图。

## 未覆盖风险

本轮不验证真实游戏切人时间、角色技能、鼠标、全键盘支持或多角色并发；切人时序是作者编写规则，真实输入验证仍留给后续独立阶段。
